# DES

## Basics

* Plaintext -> 64 bits 
* Ciphertext -> 64 bits
* Key -> 64 bits 

## Steps

* IP - Initial Permutation
* Right & Left Plain Text 
* 16 rounds
* Each round -> key transformation, permutation, S-box, P-box permutation, swap/XOR
* Final permutation
* 5 modes -> ECB, CBC, CFB, OFB, CTR
